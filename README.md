# Coffee Shop System

A web-based coffee shop management system built with **Python Flask**.  
This project allows users to browse the menu, add items to a cart, and simulate orders.

## Features

- View coffee menu and product details
- Add items to a cart
- Simple order management
- Responsive web interface
- Built using Flask, HTML, CSS, and JavaScript

## Technologies Used

- Python 3.x
- Flask
- HTML / CSS / JavaScript
- Jinja2 templates
